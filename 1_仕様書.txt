📘 ReGenesis Engine – Full Eco Simulation
A>B>C>A ecosystem + mutation + predation + toxin + E/F-type evolution engine

🌍 Overview
ReGenesis Engine は、人工生命をテーマにしたセルオートマトン型の生態系シミュレーションです。
個体はランダムに動き、捕食し、突然変異し、進化し、死に、そして環境そのものも変動します。

本プロジェクトは、
「生命はどこまで複雑さを獲得できるのか？」
という問いに対する、ひとつの実験的アプローチです。

🧬 Features
🔁 1. Rock–Paper–Scissors 型の捕食関係（A > B > C > A）
A：捕食者

B：スカベンジャー

C：吸収型捕食者

G：生産者（基礎資源）

この三すくみ構造が、世界の基本的な動態を作ります。

🧪 2. 突然変異と形質進化
個体は分裂時に突然変異し、別タイプへ変化することがあります

トゲ（spikes）、伸長率（elongation）、色相（colorShift）などの形質が継承・変異

Aタイプは経験値に応じて進化し、外見と能力が変化

死亡時に“記憶”を地面へ残し、次世代がそれを吸収して強化される

🧓 3. Eタイプ（Elder/Worm）
Cタイプが捕食を極めると昇華して誕生

長寿・高耐久

トゲ継承

死亡時に周囲の土地を肥沃化する

群れ（Cluster）との戦闘にも関与

🧠 4. Fタイプ（集合知生命体）
Eタイプが密集し、廃棄物が蓄積すると誕生

世界の“文明的進化”を象徴する存在

🧩 5. Cluster（群れ）システム
Cタイプが3体以上近接すると群れを形成

群れは単体より強力で、周囲の個体を排除

群れには寿命があり、時間とともに崩壊

🌋 6. 環境イベント
粒子降雨（軽度の資源増加）

天変地異（大規模な資源変動）

毒気候の発生

世界リセットイベント

📊 7. リアルタイム統計チャート
G / A / B / C / E / EvA（進化A）の個体数をプロット

1000ターン分のデータを保持

軽量なCanvas描画でリアルタイム更新

📜 Philosophy
このシミュレーションは、
「生命は単純なルールからどこまで複雑さを生み出せるか」
というテーマを追求しています。

形質の蓄積

記憶の継承

群れの形成

環境との相互作用

死と再生

これらが絡み合い、予測不能な“生態系の物語”が生まれます。
