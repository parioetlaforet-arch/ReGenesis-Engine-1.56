 ReGenesis Engine – Full Eco-SimulationA>B>C>A Ecosystem + Mutation + Predation + Toxin + E/F-type Evolution Engine🌍 OverviewReGenesis Engine is a cellular automaton-based ecosystem simulation themed around artificial life. Entities move randomly, prey on each other, mutate, evolve, and die within an environment that is constantly in flux.This project represents an experimental approach to a single question:"To what extent can life acquire complexity?"🧬 Features🔁 1. Rock–Paper–Scissors Predation (A > B > C > A)The world's core dynamics are driven by a three-way deadlock:Type A: PredatorType B: ScavengerType C: Absorptive PredatorType G: Producer (Basic Resource)🧪 2. Mutation & Trait EvolutionDivision Mutation: Entities may mutate into different types during cell division.Inheritance: Traits such as Spikes, Elongation, and ColorShift are passed down and modified.Experience-Based Evolution: Type A evolves based on "experience points," altering both appearance and capabilities.Memory Inheritance: Upon death, entities leave "memories" in the soil, which are absorbed by the next generation to grant enhancements.🧓 3. Type E (Elder / Worm)Born when a Type C reaches the pinnacle of predation and undergoes "sublimation."Longevity & High Durability: Extreme survival capabilities.Spike Inheritance: Retains offensive traits.Soil Enrichment: Fertilizes the surrounding land upon death.Combat: Capable of engaging with large Clusters.🧠 4. Type F (Collective Intelligence)A manifestation of the world's "civilizational evolution," emerging when Type E density is high and waste products accumulate.🧩 5. Cluster SystemFormation: Triggered when three or more Type C entities are in close proximity.Dominance: Clusters are significantly more powerful than individuals and actively eliminate surrounding entities.Decay: Clusters have a finite lifespan and will eventually collapse.🌋 6. Environmental EventsParticle Rainfall: Subtle resource increase.Cataclysm: Massive resource fluctuations.Toxic Climate: Outbreak of poisonous atmospheric conditions.World Reset: Total extinction and renewal events.📊 7. Real-Time Statistical ChartingPlots population counts for G / A / B / C / E / EvA (Evolved A).Retains data for up to 1,000 turns.Real-time updates via lightweight HTML5 Canvas rendering.🎮 ControlsActionDescriptionStartBegin the simulation.StopPause the process.RestartReset and regenerate the world.God’s MischiefTrigger special event (Descent of the Primordial Type E).Canvas Dbl-ClickToggle Screensaver Mode (Full-screen rendering).⚙️ ParametersAdjust the following via the top sliders:Initial Population / Simulation SpeedA/B Mutation RatesToxicity Coefficient / Max Toxic ConcentrationNutrient Fertility / Density-Dependent MortalityType G Spontaneous Generation Rate🏗️ Technical NotesRendering: Lightweight HTML5 Canvas API.Engine: Pure Vanilla JavaScript (No external libraries required).Grid: 160×160 Cell Map.Logic Loop: Each turn executes:Movement → Predation → Death Check → Division → Mutation → Environmental Flux.📜 PhilosophyThis simulation pursues the theme:"How much complexity can emerge from simple rules?"The accumulation of traits, the inheritance of memory, the formation of clusters, interaction with the environment, and the cycle of death and rebirth...These elements intertwine to create an unpredictable "narrative of the ecosystem."📦 How to RunDownload or clone this repository.Open index.html in any modern web browser.Press the Start button.(No additional setup or dependencies required.)📝 License[MIT License / or your choice]🙌 AuthorCreated by M.
