\# ReGenesis Engine – Full Eco-Simulation  

A > B > C > A Ecosystem + Mutation + Predation + Toxin + E/F-type Evolution Engine  

ReGenesis Engine ver1.56 / 2026  

Created by M. Morimoto



\---



\## 🌍 Overview

ReGenesis Engine is a \*\*cellular automaton–based ecosystem simulation\*\* themed around artificial life.  

Entities move randomly, prey on each other, mutate, evolve, and die within an environment that is constantly in flux.



This project represents an experimental approach to a single question:



> \*\*"To what extent can life acquire complexity?"\*\*



\---



\## 🧬 Features



\### 🔁 1. Rock–Paper–Scissors Predation (A > B > C > A)

The world's core dynamics are driven by a three-way deadlock:



\- \*\*Type A: Predator\*\*  

\- \*\*Type B: Scavenger\*\*  

\- \*\*Type C: Absorptive Predator\*\*  

\- \*\*Type G: Producer (Basic Resource)\*\*



\---



\### 🧪 2. Mutation \& Trait Evolution

\- Entities may mutate into different types during cell division  

\- Traits such as \*\*Spikes\*\*, \*\*Elongation\*\*, and \*\*ColorShift\*\* are inherited and modified  

\- Type A evolves based on accumulated “experience points,” altering both appearance and capabilities  

\- Upon death, entities leave \*\*“memories”\*\* in the soil, which are absorbed by the next generation to grant enhancements  



\---



\### 🧓 3. Type E (Elder / Worm)

Born when a Type C reaches the pinnacle of predation and undergoes “sublimation.”



\- Longevity and high durability  

\- Spike inheritance  

\- Enriches surrounding soil upon death  

\- Capable of engaging with large Clusters  



\---



\### 🧠 4. Type F (Collective Intelligence)

A manifestation of the world’s \*\*civilizational evolution\*\*, emerging when Type E density is high and waste products accumulate.



\---



\### 🧩 5. Cluster System

\- Triggered when three or more Type C entities are in close proximity  

\- Clusters are significantly more powerful than individuals  

\- Actively eliminate surrounding entities  

\- Have a finite lifespan and eventually collapse  



\---



\### 🌋 6. Environmental Events

\- Particle Rainfall (subtle resource increase)  

\- Cataclysm (massive resource fluctuations)  

\- Toxic Climate  

\- World Reset (total extinction and renewal)  



\---



\### 📊 7. Real-Time Statistical Charting

\- Plots population counts for \*\*G / A / B / C / E / EvA (Evolved A)\*\*  

\- Retains up to 1,000 turns of data  

\- Lightweight HTML5 Canvas rendering  



\---



\## 🎮 Controls



| Action | Description |

|--------|-------------|

| \*\*Start\*\* | Begin the simulation |

| \*\*Stop\*\* | Pause the process |

| \*\*Restart\*\* | Reset and regenerate the world |

| \*\*God’s Mischief\*\* | Trigger special event (Descent of the Primordial Type E) |

| \*\*Canvas Double-Click\*\* | Toggle Screensaver Mode (full-screen rendering) |

\## ⚙️ Parameters

Adjustable via the top sliders:



\- Initial Population  

\- Simulation Speed  

\- A/B Mutation Rates  

\- Toxicity Coefficient / Max Toxic Concentration  

\- Nutrient Fertility  

\- Density-Dependent Mortality  

\- Type G Spontaneous Generation Rate  



\---



\## 🏗️ Technical Notes

\- Lightweight HTML5 Canvas rendering  

\- Pure Vanilla JavaScript (no external libraries)  

\- 160×160 cell map  

\- Each turn executes:  

&#x20; \*\*Movement → Predation → Death Check → Division → Mutation → Environmental Flux\*\*



\---



\## 📜 Philosophy

This simulation pursues the theme:



> \*\*"How much complexity can emerge from simple rules?"\*\*



The accumulation of traits, inheritance of memory, formation of clusters, interaction with the environment, and the cycle of death and rebirth intertwine to create an unpredictable \*\*“narrative of the ecosystem.”\*\*

1\. Download or clone this repository  

2\. Open `index.html` in any modern web browser  

3\. Press the \*\*Start\*\* button  



No additional setup or dependencies required.
