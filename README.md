🪐 3D N-body Simulation — 完全版 🪐
HTML5 Canvas と純粋な JavaScript (Vanilla JS) のみで構築された、フルスクラッチの3次元N体（多体系）重力シミュレーターです。
外部の3Dライブラリ（Three.js 等）を一切使用せず、厳密なパースペクティブ投影と三次元幾何学配置ロジックを独自に実装し、Canvas 2D 上に美しい星々の軌跡を描き出します。

🌌 概要
本プロジェクトは、万有引力の法則に基づき、数百もの天体が互いに引き合い、混沌（カオス）と秩序の狭間で織りなす宇宙の営みをリアルタイムに再現するシミュレーターです。
単なる物理計算にとどまらず、時空の相転移を感じさせる「宇宙管制盤（プリセットシステム）」や、物理的な規律を視覚化する様々な表示連動機能を備えています。

✨ 主な機能
1. バニラJSによる3Dグラフィックスエンジン
等倍パースペクティブ投影 (project3D): ニア・クリッピング（近接透過判定）を備えた厳密な3D→2D画面変換。

ピボット・シフトカメラ (rotate3D): ターゲット天体（太陽など）を中心に、世界を美しく旋回・巡航させるフリー3DカメラおよびOrbitモードを搭載。

2. 宇宙管制盤（インジェクション・プリセット）
物理層とUI層のパラメータを同時に書き換える、4つの初期宇宙プリセットを配線済：

🌀 安定軌道（基準宇宙・鑑賞モード）: 太陽に能動的な初期速度を与え、美しい螺旋・円軌道を鑑賞するモード。

💥 カオス崩壊（物理破綻検出）: 高密度・高重力下でシステムが破綻していく様を観測するモード。

🌌 拡散テスト（境界条件テスト）: 天体が宇宙の彼方へエスケープしていく挙動を検証するモード。

💻 描画負荷（計算限界相）: 計算・描画の極限に挑む高負荷シミュレーション。

3. 特異天体とカオス要素の動的生成
逆回り天体: 10%の確率で周囲の星々と逆方向に公転を始める天体群を生成し、宇宙にドラマを生み出す。

極限スイングバイ彗星 (addComet): 太陽への「かすりUターン（極至近落下）」または「大楕円スイングバイ」を行う計算軌道を搭載。

異分子『Goliath』 (spawnGoliathForce): 1%の確率（またはボタンによる強制召喚）で誕生する、太陽と同等の質量を持つ特異点。宇宙の重力場を一瞬で歪めます。

4. 物理の規律・視覚化システム
均一グリッド（Uniform Grid）衝突判定: 高速な近接遭遇判定と、重力大爆発を防ぐ軟化係数（eps2: 64）の調律。

ベクトル・重心表示: バリセンター（共通重心）表示、および角運動量ベクトルのリアルタイムネオンエメラルド点灯表示の切り替えに対応。

🛠 技術スタック
Language: JavaScript (ES6+)

Rendering: HTML5 Canvas API (2D Context)

Style: Cyberpunk / Minimal UI (HTMLスライダー・ボタン完全同期)

Dependencies: なし（完全なる独立系スクリプト）

🕹 操作方法とUIコントロール
マウス左ドラッグ: カメラの回転（ジンバルロック防止の角度制限付き）。

マウス右ドラッグ: カメラの平行移動（オフセット調整）。

マウスホイール: 宇宙のズームイン / ズームアウト（最大200倍のワイドレンジ）。

キーボードショートカット:

r / t / y / u: 太陽位置・追従カメラのコマンド（固定・解除・追従・センター）。

i / o / p: 軌跡レンダリングフィルタ（彗星、惑星、太陽の軌跡切り替え）。

j / k / l: カメラプリセット（赤道面、極、俯瞰オーバービュー）への一発切り替え。
